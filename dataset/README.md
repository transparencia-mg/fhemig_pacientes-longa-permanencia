Conjunto de dados referentes a unidade de medida que representa a proporção de pacientes que excedem o tempo de internação padrão (30 dias). Complementa o indicador de tempo médio de permanência, identificando casos que requerem otimização do plano de cuidado e contribuindo para a otimização da ocupação dos leitos na rede pública.

Termos relacionados: Longa Permanência; Média de Permanência.
